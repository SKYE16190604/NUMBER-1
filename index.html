INTODUCTION TO HTML

HTML stands for hyper text mark-up language. It is the standard mark-up language used to create webpages. HTML is used to define the structure and content of webpages, using elements like headings, paragraphs, images, links, forms, tables and more. HTML is not a programming language, but rather a mark-up language that tells web browsers how to display content. It is a crucial part of web development, working alongside CSS(for styling) and JavaScript(for interactivity).

STEPS INVOLVED IN HTML

There are certain steps involved in the creation of webpages using HTML as the mark-up language, these steps include the following:

  1. Planning and design: Determine the purpose and target audience, and sketching wireframe or mockup of its layout and design
  2. Writing HTML code: Use a text editor or IDE to write HTML elements, beginning with the basic structure of an HTML document which includes the , , , and elements
  3. Creating a basic structure: The basic structure of an HTML document is comprised of the declaration, the root element, the element, which contains metadata about the document and the element, which contains the content of the HTML document
  4. Adding content: You then add content to your page using HTML elements, such as headings, paragraphs, images, links, lists, tables, and forms, and use attributes to provide additional information about these elements
  5. Use of HTML elements: HTML elements are used to define the structure and content of your page, and include elements such as headings(<h1>-<h6>), paragraphs(<p>), images(<img>), links(<a>), lists(<ul>,<ol>,<li>), tables(<table>,<tr>,<td>), and forms(<form>,<input>,<textarea>,<button>).
  6. Adding attributes: Attributes are used to provide additional information about elements, and can include global attributes sych as id, class, and style, as well as attributes specific to certain elements such as src for images and href for links
  7. Testing and validation: After writing the HTML code, you validate and test it using tools like W3C validator to check for errors and test it with different web browsers and on different devices to ensure compatibility
  8. Refining and iteration: Making adjustments and correcting errors on your HTML code based on testing and feedback and continuously improving and refining your page's design and functionality

    9. HTML ELEMENTS

    An HTML element is defined by a start tag, some content, and an end tag:(content)
    Examples of some HTML elements:
    <h1>My First Heading</h1>
    <p>My First Paragraph</p>

    NESTED HTML DOCUMENTS

    HTML elements can be nested(this means that elements can contain other elements). All HTML documents consists of nested HTML elements.
    The following example contains four HTML elements(<html> <body> <h1> <p>): 

    <!DOCTYPE html>
<html>
<body>
<h1>My first heading</h1>
<p>My first paragraph</p>
</body>
</html>
    Example Explained
    The <html> element is the root element and it defines the whole HTML document. It has a start tag <html> and an end tag </html>. Then inside the <html> element there is a <body> element:
    <body>
    <h1>My First Heading</h1>
    <p>My First Paragraph</p>
    </body>
    The <body> element defines the document's body. It has a start tag <body> and an end tag </body>. Then, inside the <body>, there are two more elements:<h1> and <p>
    The <h1> element defines a heading. It has a start tag <h1> and end tag </h1>
    The <p> element defines a paragraph. it has a start tag <p> and an end tag </p>.

    HTML ATTRIBUTES

    • All HTML elements can have attributes
    • Attributes provide additional information about elements
    • Attributes are always specified in the start tag
    • Attributes usually come in name/value pairs like: name:"value"
